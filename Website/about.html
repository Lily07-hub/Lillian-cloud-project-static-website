About This Project

This is a final project for my Cloud Computing course. The goal was to deploy a static website using cloud automation tools.

I used AWS S3 for hosting, a Bash script for provisioning, and GitHub Actions for CI/CD.

All code is open and fully automated—push to GitHub, and the site updates instantly!
